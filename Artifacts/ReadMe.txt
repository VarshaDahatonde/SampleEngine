
This toolkit is one-click solution for all the customers, partners and other Tosca users.
Below are the offerings in the version 1.0:-
1.SETs
	a)	Buffer Instance
	b)	Clear Browser
	c)	Close Browser
	d)	HelloWorld
	e)	Open URL
	f)	Start Program
2.Custom Controls
	a)	Accordion To Link
	b)	Combo With Remote Data
	c)	Div To Button
	d)	Folder Structure To Tree
	e)	Table with embedded dropdown
	f)	Toggle – Checkbox
3.Add-Ons
	a)	Ribbon Task – DLL Replacement 
	b)	Drag Drop – Test Cases from Modules
	c)	Context Menu Task – Show all engines used in a module folder
